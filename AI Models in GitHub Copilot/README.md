## How to Use

1. **Which models are available?** Start with document 01 for a full catalog.
2. **Which model should I use?** Document 02 has task-based recommendations.
3. **How much does it cost?** Document 03 breaks down pricing and credits.
4. **Just want it to work?** Document 04 explains the Auto selection feature.
5. **Admin configuring access?** Document 05 covers org-level policies.
